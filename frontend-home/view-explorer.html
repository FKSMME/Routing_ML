🔍 SQL View Explorer

MSSQL 뷰를 선택하고 홈 대시보드에 표시할 데이터를 설정하세요

데이터베이스 연결이 필요합니다

MSSQL 서버에 연결하지 못했습니다. 환경 변수와 네트워크(VPN 등)를 확인한 뒤 다시 시도해 주세요.

  • MSSQL_SERVER, MSSQL_DATABASE, MSSQL_USER 설정을 점검합니다.
  • SQL Server ODBC 드라이버가 설치되어 있는지 확인합니다.
  • 사내 네트워크 또는 VPN 연결이 필요한 경우 접속 상태를 확인합니다.

최근 오류 코드: 없음

Step 1: MSSQL View 선택
Step 2: 샘플 데이터 미리보기
Step 3: 컬럼 설정 (드래그로 순서 변경 가능)

활성화하면 각 행에 체크박스가 표시되고, 체크한 사용자 이름이 기록됩니다.

✓ 설정이 저장되었습니다! 홈 페이지를 새로고침하면 데이터가 표시됩니다.
