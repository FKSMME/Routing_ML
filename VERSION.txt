v5.1.0 (2025-10-15) - 라우팅 자동생성 시스템
=============================================
MAJOR UPDATE - Node-based workflow visualization, compact 4-column design, single-row performance charts

New Highlights:
- ✓ 프로젝트명 변경: "MCS Server Dashboard" → "라우팅 자동생성 시스템 모니터"
- ✓ 노드 기반 워크플로우 시각화: 폴더 선택 → 서버 시작 → 서버 정지 → 캐시 정리
- ✓ 4-컬럼 서비스 그리드 레이아웃으로 공간 효율성 극대화
- ✓ 단일 행 성능 차트 (CPU, 메모리, 응답시간, 디스크)
- ✓ 표준 Windows 탭 디자인 (선택된 탭 음영 효과)
- ✓ 마우스 휠 스크롤링 지원
- ✓ 컴팩트한 카드 디자인으로 더 많은 정보를 한 화면에 표시

Build Types:
- Portable EXE: Single file executable (RoutingML_AutoGen_v5.1.0_Portable.exe)
  - Build command: deploy\build_v5_1_complete.bat
  - Use case: Quick deployment, testing, portable usage

- Installer Package: Directory-based installation
  - Directory: dist\RoutingML_AutoGen_v5.1.0\
  - Executable: RoutingML_AutoGen.exe
  - Use case: Permanent installation, faster startup

Bug Fixes:
- Fixed admin checkbox not working in user approval
- Fixed background color consistency across all tabs
- Improved service status detection

Technical Details:
- Python 3.11+
- Tkinter GUI framework
- psutil for system monitoring
- PyInstaller for executable packaging

Dependencies:
- psutil
- tkinter (built-in)
- urllib (built-in)
- threading (built-in)

System Requirements:
- Windows 10/11 (x64)
- 4GB RAM minimum
- 100MB disk space
- Network access to backend services

Installation:
1. Download the appropriate build (portable or installer)
2. For portable: Run RoutingMLMonitor_v4.1.0.exe directly
3. For installer: Extract folder and run the .exe inside
4. Select your project folder on first run
5. Click "Start All" to launch services

Usage:
- Service Monitor tab: View and control all services
- Real-time Performance tab: Monitor system resources
- Backend Logs tab: View and filter backend logs
- User Management tab: Approve/reject user registrations
